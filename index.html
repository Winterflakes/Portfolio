Task Manager Using LARAVEL

This project is entirely built on the LARAVEL framework and is a part of my HONOR's final project for the semester 4th. To design web pages, HTML, and CSS were used. Also, this was made using PHP. The database connections and SQL were managed using the SQL Work Bench. The main focus is to demonstrate knowledge about working with the framework and PHP hence styling was done in a minimalist approach. The features also demonstrate the implementation of the CRUD operations.

Key Features of this Web App:

1. Task Listing: Each task is displayed with its title, and the user can quickly scan through their to-do list.
2. Task Details: By clicking on a task from the list, users can view detailed information about the task, including its title and description.
3. Adding New Tasks: Users can add new tasks to their to-do list using the "Create New Task" feature also by providing a title and an optional description.
4. Updating Tasks: users can modify the task's title and description using the "Edit Task" feature.
5. Task Deletion: Completed or unnecessary tasks can be removed from the list using the "Delete" feature.
This project deepened my understanding of how routes, controllers, models, and views work together. And I also understood how can you change, access, modify or control routes and their content. This counts as an outcome of the HONORs program for the 4th semester.

Chrome Extension Development - 2048 game

Transform your browser into an interactive gaming zone with the "2048 Game" Chrome extension! Built using HTML, CSS, and JavaScript, this extension lets you enjoy the classic 2048 puzzle game directly within your browser. Sharpen your strategic thinking and logical skills as you swipe to merge tiles and reach the elusive 2048 tile. Features: Seamless Integration: Play the addictive 2048 game without leaving your browser tab. The extension's content script smoothly embeds the game interface into web pages. Interactive Gameplay: Swipe your way to success using the arrow keys. Merge tiles to create higher-valued tiles and challenge yourself to achieve the legendary 2048 tile. Dynamic UI: The extension's user interface adapts flawlessly to your browser's theme, offering a cohesive and enjoyable experience. Responsive Design: The game is designed to work flawlessly on various screen sizes, ensuring an optimal gaming experience on both desktop and mobile. Service Worker Management: The extension's service worker handles background tasks, enhancing performance and responsiveness while managing data caching. Manifest Structure: The extension's manifest file (Manifest v3) is meticulously configured to ensure proper permissions, resources, and interactions with the browser. Score Tracking: Keep track of your progress with an integrated score system. Challenge yourself to beat your high score and achieve mastery of the game. Skills Demonstrated: HTML, CSS, JavaScript: Developed the game's interface, interactions, and logic using these core web technologies. Manifest v3: Created a well-structured manifest file to manage extension permissions, resources, and interactions. Service Worker: Implemented a service worker to manage background tasks, improve performance, and enhance the gaming experience. Responsive Design: Ensured the game's usability and appeal across various devices and screen sizes. DOM Manipulation: Leveraged DOM manipulation techniques to dynamically update the game interface and keep players engaged. Event Handling: Utilized event listeners to capture user input and trigger game actions based on arrow key presses. Problem Solving: Designed and implemented the game's logic, including tile merging, movement, and win/lose conditions.
